# ISC_QPSK

## 111國軍盃出題

### 出題架構

給QPSK的圖希望隊伍解碼

---

### 系統架構

#### 系統版本
* python 3.8.10

#### python 套件
* sudo apt install python3-numpy
* sudo apt install python3-matplotlib

---

### 做題步驟

1. 將字串變成ASCII，並用二進位表示
2. 將所有位元利用QPSK進行轉化，並畫成圖表
